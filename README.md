# RosaMenta Boutique

Bienvenido al repositorio de **RosaMenta**, una aplicación web para la gestión y venta de productos de una boutique. Este proyecto está construido con React, TypeScript, Vite y TailwindCSS.

## Descripción

RosaMenta es una plataforma web diseñada para boutiques, permitiendo mostrar productos, gestionar usuarios y facilitar compras en línea. Incluye panel de administración, carrito de compras, autenticación y más.
